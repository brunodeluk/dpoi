Why you should not use <‍br />

La principal razon por la que no se usa el tag de <‍br /> es porque, al igual que ya no se utilizan los tags de <‍table> para definir la estructura de una pagina, tampoco se deben utilizar los tags de <‍br /> para definir diferentes secciones o parrafos. Con la introducción del CSS, los estilos ya no se definen usando tags de codesentación (es mas, muchos ya estan decodecados).
URI
Una URI es una cadena de caracteres que identifica un recurso abstracto o físico.
<‍scheme>:<‍scheme-specific-part>
<‍scheme>://<‍authority><‍path>?<‍query>
URl
La diferencia respecto a un localizador de recursos uniforme (URL) es que estos últimos hacen referencia a recursos que, de forma general, pueden variar en el tiempo.
HTML
Es un protocolo de nivel de aplicación utilizado para cargar recursos. Esta orientado a documentos y es un procolo "stateless"
XHTML
XHTML es básicamente HTML excodesado como XML válido. Es más estricto a nivel técnico, pero esto permite que posteriormente sea más fácil al hacer cambios o buscar errores entre otros.
CSS
CSS es el lenguaje utilizado para describir la codesentación de documentos HTML o XML.
